<h1 align="center"> CHALLENGE LITERALURA * CATÁLOGO DE LIVROS * </h1>

O LiterAlura é um challenge que foi desenvolvido para testar as habilidades adquiridas durante os cursos da Alura de java spring boot
que foram disponibilizados para a turma 6 do Oracle One. 

 O desafio é construir um catalogo de livros que utiliza uma API externa chamada Gutendex, que fornece uma biblioteca online de e-books gratuitos.

Essa aplicação oferece interação textual (via console) com os usuários para fazer uma busca de livros contendo cinco opções de interação
que posteriormente serão salvos em um banco de dados.  

<h1 align="center">* FUNCIONALIDADES * </h1>

- Buscar livro pelo título
- Listar livros registrados
- Listar autores registrados
- Listar autores vivos em um determinado ano
- Listar livros em um determinado idioma


<h1 align="center"> TECNOLOGIAS UTILIZADAS </h1> 

- Java 17
- Spring Boot 3.2.6
- Spring Data JPA
- PostgresSQL
- Jackson(deserialização JSON)
- Maven
